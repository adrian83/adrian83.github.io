Decorators in Python

January 12, 2019 in python, decorator, decorators

Decorators are one of my favorite Python features. By using them we can alter functions and methods. From decorator we have access to method / function arguments as well as returned value. We don’t even have to execute decorated function.

Decorators can be implemented in two ways: as a function or as a class. Both types of decorators offer almost the same functionality.

First let’s see code that will be used in all examples: (1) Function that will return text information about wrapped function (name and given parameters). This function will be used internally by every presented decorator. (2) Function that will be ‘decorated’ by every presented decorator.

Summary

Decorators are very useful Python feature which is used heavily in almost all Python frameworks as well as in Python standard library (@classmethod, @staticmethod). That’s why it is very important to understand how to create and use them.
